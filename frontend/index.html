Instructions

Before using the website

1. Use with a caregiver if possible

The questions are easier when answered together with a family member or trusted caregiver.

2. Answer based on current daily life

Choose the answer that best reflects how the person is doing now, not how they were years ago.

3. This is a support tool, not a diagnosis

The AI estimate helps flag risk patterns. A doctor or memory clinic is needed for diagnosis.

4. Results stay on this device

Screening results and daily game scores are stored in this browser so progress can be tracked privately.
